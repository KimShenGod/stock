# 数据源配置

本项目支持两种数据获取方式：本地通达信数据导出和网络数据获取。

## 数据来源概览

| 数据类型 | 本地来源 | 网络来源 | 更新方式 |
|---------|---------|---------|---------|
| 日线数据 | `/vipdoc/sh(sz)/lday` | baostock/pytdx | 通达信手动/自动更新 |
| 财务报告 | `/vipdoc/cw` | baostock | 代码自动维护 |
| 股本变迁 | `/T0002/hq_cache/gbbq` | baostock | 通达信自动更新 |
| 指数数据 | `/vipdoc/sh(sz)/lday` | pytdx | 通达信更新 |

## 用户配置文件

编辑 `user_config.py` 配置本地路径和数据源：

```python
debug = False  # 是否开启调试日志

tdx = {
    'tdx_path': 'c:/new_tdx64',  # 通达信安装目录
    'csv_lday': './TDXdata/lday_qfq',  # CSV日线数据目录
    'pickle': './TDXdata/pickle',  # Pickle格式数据目录
    'csv_index': './TDXdata/index',  # 指数数据目录
    'csv_cw': './TDXdata/cw',  # 财务报告目录
    'csv_gbbq': './TDXdata',  # 股本变迁目录
    'pytdx_ip': '218.6.170.55',  # pytdx服务器IP
    'pytdx_port': 7709,  # pytdx服务器端口
}

index_list = [
    'sh000001.day',  # 上证指数
    'sh000300.day',  # 沪深300
    'sz399001.day',  # 深成指
    'sz399006.day',  # 创业板指
]
```

## 本地数据导出

### 日线数据导出

运行 `readTDX_lday.py` 导出通达信日线数据：

```bash
# 正常更新（增量追加）
python readTDX_lday.py

# 强制重新生成全部数据
python readTDX_lday.py del
```

导出的数据自动进行前复权处理，与通达信软件显示价格一致。

### 财务报告导出

运行 `readTDX_cw.py` 导出专业财务数据：

```bash
python readTDX_cw.py
```

财务数据自动判断是否需要更新，仅下载新增部分。

### 股本变迁导出

股本变迁数据随财务报告一起导出，用于计算前复权价格。

## 网络数据获取

### 使用 fetch_market_data.py

无需通达信软件，直接从网络获取数据：

```bash
python fetch_market_data.py
```

数据来源：
- **日线数据**：baostock（免费开源）
- **实时行情**：pytdx（通达信服务器）
- **财务数据**：baostock

### 使用 get_eastmoney_enhanced.py

通过 akshare 获取东方财富数据：

```bash
python get_eastmoney_enhanced.py
```

支持市场：SH、SZ、BJ、HK、US

## 数据更新流程

### 盘后更新流程

1. 16:00后通达信自动更新日线数据
2. 运行 `readTDX_cw.py` 更新财报
3. 运行 `readTDX_lday.py` 导出日线

### 盘中数据获取

盘中选股时自动获取实时行情：

```bash
python xuangu.py
```

程序会自动：
- 加载本地历史数据
- 获取当日实时行情
- 合并数据进行前复权计算

## 数据文件格式

### CSV日线格式

| 字段 | 类型 | 说明 |
|-----|-----|-----|
| date | str | 日期 YYYY-MM-DD |
| code | str | 股票代码 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| volume | float | 成交量 |
| amount | float | 成交额 |
| 换手率 | float | 换手率% |
| 流通市值 | float | 流通市值 |

### Pickle格式

pickle格式存储完整的DataFrame，读取速度更快，推荐用于回测。

## pytdx服务器配置

### 自动选择最优服务器

运行 `pytdx_best_ip.py` 自动测速选择最快服务器：

```bash
python pytdx_best_ip.py
```

结果保存在 `best_ip.json`。

### 手动指定服务器

编辑 `user_config.py`：

```python
tdx = {
    'pytdx_ip': '218.6.170.55',
    'pytdx_port': 7709,
}
```

常用服务器：
- 218.6.170.55:7709（福州）
- 115.238.90.165:7709（杭州）
- 124.160.88.21:7709（上海）
