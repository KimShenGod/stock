# 策略系统

本项目采用策略注册模式，支持自定义选股策略的组合使用。

## 策略注册机制

```python

    # 策略逻辑
```


## 策略实现示例

### 高开涨停策略

```python
@register_strategy("高开涨停")
def high_open_limit_up_strategy(df, start_date='', end_date='', mode=None):
    close_price = df['close']
    open_price = df['open']
    high_price = df.get('high', close_price)
    
    prev_close = close_price.shift(1)
    prev2_close = close_price.shift(2)
    prev_high = high_price.shift(1)
    
    # T-1日涨停
    prev_limit_up_price = prev2_close * 1.1
    prev_limit_up = prev_high >= prev_limit_up_price * 0.995
    
    # T日高开2%以上
    today_gap_up = open_price >= prev_close * 1.02
    
    signal = prev_limit_up & today_gap_up
    return signal.fillna(False)
```

### MACD金叉策略

```python
@register_strategy("MACD周线金叉")
def macd_weekly_golden_cross_strategy(df, start_date='', end_date='', mode=None):
    close_price = df['close']
    
    # MACD计算
    ema12 = close_price.ewm(span=12, adjust=False).mean()
    ema26 = close_price.ewm(span=26, adjust=False).mean()
    dif = ema12 - ema26
    dea = dif.ewm(span=9, adjust=False).mean()
    
    # 金叉信号
    signal = (dif > dea) & (dif.shift(1) <= dea.shift(1))
    return signal.fillna(False)
```

## TDX公式函数

`func_TDX.py` 提供通达信公式语言的Python实现：

| 函数 | 说明 | 用法 |
|-----|-----|-----|
| REF | 前N日数据 | `REF(close, 5)` |
| MA | 移动平均 | `MA(close, 5)` |
| SMA | 简单移动平均 | `SMA(close, 5)` |
| HHV | N日最高值 | `HHV(high, 20)` |
| LLV | N日最低值 | `LLV(low, 20)` |
| COUNT | 统计满足条件次数 | `COUNT(condition, 5)` |
| EXIST | N日内存在 | `EXIST(condition, 5)` |
| CROSS | 上穿判断 | `CROSS(line1, line2)` |
| BARSLAST | 上次满足距今 | `BARSLAST(condition)` |

## 策略组合配置

在 `config.yml` 中配置策略组合：

```yaml
strategies:
  # 策略组合为AND关系（同时满足所有条件）
  default: ["周线MACD区间", "小市值"]
  combo_1: ["前日涨停", "小市值"]
  combo_2: ["高开涨停", "换手率"]
  combo_3: ["今日涨停", "小市值"]
```

## 策略信号保存

运行 `celue_save.py` 批量保存策略信号：

```bash
# 正常更新（增量）
python celue_save.py

# 强制重新生成
python celue_save.py del

# 单进程执行
python celue_save.py single
```

生成文件：
- 各股票pickle文件中添加策略信号列
- `celue汇总.csv` 包含所有买卖信号
