# 选股功能

本项目支持盘中选股和盘后选股两种模式。

## 选股入口

运行 `xuangu.py` 进行选股：

```bash
python xuangu.py
```

## 选股模式

### 盘后选股

**时间**：周一~周五 16:01~8:59，或周末

**数据源**：本地历史数据（readTDX_lday.py生成）

**流程**：
1. 确保通达信已更新当日数据（16:00自动更新）
2. 运行 `readTDX_lday.py` 导出最新数据
3. 运行 `xuangu.py` 选股

### 盘中选股

**时间**：周一~周五 9:00~16:00

**数据源**：本地历史数据 + 当日实时行情

**流程**：
1. 程序自动获取实时行情
2. 合并到历史数据（内存操作）
3. 自动判断是否需要前复权调整
4. 执行选股策略

## 选股配置

### 日期设置

在 `xuangu.py` 中设置日期范围：

```python
# 使用当天数据
start_date = ''
end_date = ''

# 指定日期区间
start_date = '20240101'
end_date = '20241231'
```

### 策略选择

`xuangu.py` 自动调用 `celue.py` 中定义的策略：

```python
# xuangu.py 会导入并执行 celue.py 中的策略
import celue
```

确保个人策略保存在 `celue.py` 文件中。

## 选股输出

运行后显示满足条件的股票列表：

```
已选出股票列表:
['000001', '000002', '600000', ...]
```

此列表即为当日策略发出买入信号的股票。

## 自动化交易对接

### 买入对接

`xuangu.py` 最后的变量"已选出股票列表"即为可买入股票：

```python
# 在自动化交易系统中读取该列表
buy_list = get_selected_stocks()
for stock in buy_list:
    execute_buy(stock)
```

### 卖出对接

需要自行编写持仓获取逻辑：

```python
import user_config as ucfg
import pandas as pd

def check_sell_signals(holdings):
    """检查持仓股卖出信号"""
    for stock_code in holdings:
        df_stock = pd.read_csv(
            ucfg.tdx['csv_lday'] + '/' + stock_code + '.csv',
            encoding='gbk'
        )
        df_stock['date'] = pd.to_datetime(df_stock['date'])
        df_stock.set_index('date', inplace=True)
        
        # 盘中获取实时数据
        if is_trading_hours():
            df_today = get_tdx_lastestquote(stock_code)
            df_stock = update_stockquote(stock_code, df_stock, df_today)
        
        # 检查卖出条件
        sell_signal = sell_strategy(df_stock)
        if sell_signal.iloc[-1]:
            trigger_sell(stock_code)
```

## 选股效率优化

### 使用celue汇总.csv

预先生成策略信号文件可大幅提升选股速度：

```bash
# 预先生成信号
python celue_save.py
```

选股时直接读取 `celue汇总.csv`，无需重新计算。

### 多进程处理

`celue_save.py` 默认使用多进程：

```bash
# 多进程（默认，效率高）
python celue_save.py

# 单进程（调试用）
python celue_save.py single
```

## 选股策略开发流程

1. 参照 `CeLue.py` 模板编写策略
2. 保存为 `celue.py`（个人策略文件）
3. 运行 `celue_save.py` 生成信号
4. 运行 `xuangu.py` 测试选股结果
5. 根据结果调整策略参数
